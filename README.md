# Decycles

Curated global directory of independent cycling creators — framebuilders, mechanics, organizers, and storytellers.

This repository contains the full source code for the Decycles web application.

## Run Locally

**Prerequisites:** Node.js (v18+)

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Configure environment variables:**
   Set your Gemini API key in a `.env.local` file:
   ```bash
   GEMINI_API_KEY=your_gemini_api_key_here
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```
